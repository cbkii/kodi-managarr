# Roadmap

Implemented foundations now include the private Kodi repository, configurable registry-driven menus, local destructive-action PIN, smart Request & Search, Arr-authoritative interactive release selection, Prowlarr health/informational context, a bounded Dashboard MVP, and Bazarr integration with Kodi's subtitle window.

Future considerations:

- P2 per-request root/profile, multi-instance and HD/4K routing.
- P3 targeted reconciliation and stale-row repair.
- P7 watched/added-age retention and optional periodic conditional deletion (tracked separately until its dedicated PR lands).
- P8 Jellyseerr/Overseerr request-manager integration.
- P10 expanded sanitised diagnostic-report functionality.
- Dashboard expansion: recent imports/history, commands, storage trends and configurable widgets.
- Broader Kodi/Radarr/Sonarr/Bazarr version fixtures and physical Android compatibility evidence.
- Automatic whole-series retention only after unambiguous watched-state semantics are designed.
