# Roadmap

Implemented foundations include repository-based updates, configurable menus, local PIN protection, targeted management actions and watched/age-aware movie and episode-file retention.

Future candidates:

- P2 per-request root-folder/profile or multi-instance routing.
- P3 reconciliation and stale-row repair.
- P8 Jellyseerr/Overseerr integration.
- P10 expanded diagnostic-report functionality.
- Request & Search and interactive Radarr/Sonarr release search.
- Bazarr subtitle-service and optional Prowlarr search/health integration.
- A larger dashboard with recent imports, history, storage trends, commands and configurable widgets.
- Retention policy extensions such as whole-season/series decisions, storage-pressure rules, tags/profiles and richer exclusion policies. Whole-series unattended deletion must not be added without explicit, testable watched-state semantics.
